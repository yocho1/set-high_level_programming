# Python - More Classes and Objects

## 📖 Description

This project dives deeper into **Object-Oriented Programming (OOP)** in Python. You'll learn advanced class features including:

- Class and instance attributes
- Methods (`__init__`, `__str__`, `__repr__`, `__del__`)
- Properties and encapsulation
- Class methods and static methods
- Special methods for comparison and string representation
