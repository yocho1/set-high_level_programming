# Python - Import & Modules

## 📖 Description

This project demonstrates how to use Python modules, imports, and the `if __name__ == "__main__"` guard. You'll learn how to import functions from other files and reuse code across multiple programs.

## 📁 Files

| File | Description |
|------|-------------|
| `add_0.py` | Contains the `add(a, b)` function |
| `0-add.py` | Imports `add` and prints `1 + 2 = 3` |

## 🚀 How to Run

```bash
# Make the script executable
chmod +x 0-add.py

# Run the script
./0-add.py
