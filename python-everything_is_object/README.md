# Python - Everything is Object

## 📖 Description

This project explores Python's object model, focusing on the differences between **mutable** and **immutable** objects, how variables reference objects, and how Python handles memory.

## 🧠 Key Concepts

| Concept | Description |
|---------|-------------|
| **Immutable objects** | `int`, `str`, `tuple`, `bool` |
| **Mutable objects** | `list`, `dict`, `set` |
| **`id()`** | Returns the memory address of an object |
| **`type()`** | Returns the type of an object |
| **`is`** | Checks if two objects are the same |
| **`==`** | Checks if two objects have the same value |
| **Pass by assignment** | How Python passes arguments to functions |
